Bienvenidos A Mi Página Web

¿Qué es HTML?

El Lenguaje de Marcado de Hipertexto (HTML), es un lenguaje de programación que se utiliza para estructurar y desplegar una página web y sus contenidos.

¿Qué es CSS?

CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje de programación que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita.

¿Qué es Javascript?

JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.


Tipos De Programación


Hay dos tipos de programación, front-end y back-end.

Front-end y Back-end


Frontend


El (Front-end) se encarga de lo que el usuario toca, ve e interactúa, el front-end es la presentación que le brinda al usuario, como botones, textos, colores, funciones que puedes interactuar con ella.

Backend


El (Backend) se encarga de lo que está detrás de lo que interactúas, lo que no puedes ver aunque sabes que te da un resultado, es decir, lo que hay detrás de lo que ves el cómo funciona como servidores, flask, etc.

¿Cómo pueden conectarse el frontend y el backend para trabajar juntos?


¿Cómo puede trabajar el front-end y el back-end juntos?. La forma que el front-end y el back-end trabajan juntos es gracias a una (API o Flask), una API también conocido como Interfaz de Programación de Aplicaciones sirve como "puente" entre el front-end y el back-end lo cuál los conecta a los dos.
