¿Qué es HTML?

El Lenguaje de Marcado de Hipertexto (HTML) que se utiliza para estructurar y desplegar una página web y sus contenidos.

¿Qué es CSS?

CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets) que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita.

¿Qué es Javascript?

JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.

Mi consejo personal

Tengo un mensaje para todos los Developers y los que se dedican a la programación e informática, a entender cómo funciona un sistema, un OS, un Kernel, una web, un servidor, etc. Los tutoriales en Youtube, etc, nos enseñan muchas veces lo superficial y no nos enseñan las bases y la lógica, les recomiendo bastante aprender en IAs y Google. ¿Porqué? No es que las IAs nos haga el trabajo, sino que a través de ahí podemos investigar, preguntar, aprender, podemos hacer cualquier pregunta, aprender y entender lo que no aprendemos en tutoriales y videos, por ejemplo: En tutoriales nos enseñan bucles, arrays, tipos de datos, variables, constantes, funciones, programación orientada a objetos (POO), etc. Sin embargo no nos enseñan cómo manejar un memory leak o cómo aplicar async/await (asincronía), cómo hacer cuando nuestra aplicación se queda colgada, que si por ejemplo concatenas datos a una consulta SQL tenemos el riesgo de una vulnerabilidad (a que nos tiren producción), en cambio en IAs podemos preguntar y descubrir "nuevas ramas" de conocimiento, hacer preguntas que ni siquiera se atrevieran a responder o enseñar en tutoriales, etc. Hoy les invito a que investiguen también en IAs, qué comparen información, no se queden con una sola idea u opción, usen su razonamiento, la lógica, su criterio propio, dense cuenta que todos y cada uno tenemos la capacidad de pensar, es una habilidad poderosa, ahora tienen que darse cuenta y usarla.
